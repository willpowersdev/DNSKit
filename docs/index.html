DNS for Swift · cross-platform

DNSKit

A full-featured interface to the Domain Name System.

Both client- and server-side programming are supported, giving you complete control over what goes on the wire behind a small, clean API. Resource records are native Swift value types — never stored in wire format — and the whole library runs on macOS, iOS, and Linux over SwiftNIO and swift-crypto.

resolve — swift 
import DNSKit

let client = DNSClient()
let reply = try await client.query("example.com.", .a, server: "1.1.1.1")
print(reply)  // dig-style output ↓
;; ->>HEADER<<- opcode: 0, rcode: 0, id: 4660
;; flags: qr rd ra; QUERY: 1, ANSWER: 1, AUTHORITY: 0, ADDITIONAL: 0

;; ANSWER SECTION:
example.com.  300  IN  A  93.184.216.34

The library

Everything a DNS programmer needs, and nothing they don’t.

DNSKit supports asynchronous querying and replying, incoming and outgoing zone transfers, TSIG, EDNS0, dynamic updates, and DNSSEC validation and signing. Wire codecs for all ~80 record types are generated at compile time by the @DNSRecord macro, and every encoding is verified byte-for-byte against the reference implementation.

One note carried over from the original: domain names must be fully qualified before they are sent — an unqualified name will fail to pack.

Messages

Build, pack, and parse queries and responses with full header and section control.

Records

~80 native record types, plus EDNS0, SVCB/HTTPS, and zone-file parsing.

Client & Server

Async resolver and authoritative server over UDP and TCP on SwiftNIO.

DNSSEC

RRSIG signing and validation, DS, NSEC3, TSIG, and SIG(0) via swift-crypto.

Usage

Records are values; a resolver is one call away.

Create a resource record straight from zone-file text:

records
let mx = try NewRR("example.com. 3600 IN MX 10 mail.example.com.")

// …or build it directly as a value type
let a = A(header: RRHeader(name: "example.com.", type: .a, ttl: 300),
          a: IPv4Address("192.0.2.1"))

Stand up an authoritative responder, or validate a signed answer:

server & dnssec
let mux = ServeMux()
mux.register("example.com.", myHandler)
let server = DNSServer(responder: mux)
try await server.start(host: "0.0.0.0", port: 53)

let valid = try DNSSEC.verify(rrsig: sig, rrset: [a1, a2], key: dnskey)
